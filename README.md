## Vue 3 Поиск товаров с фильтрами
Проект реализует компонент поиска товаров на Vue 3 с использованием TypeScript, Pug и FSD архитектуры.

## Функциональность
- Поиск товаров с текстовым поиском

- Фильтры по категории и цене

- Сортировка по имени, цене, категории (возр/убыв)

- Пагинация с отображением текущей страницы

- Состояние в URL - фильтры сохраняются в query-параметрах

- Дебаунс поиска

- Архитектура FSD
